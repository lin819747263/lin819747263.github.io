# 怎样才算拥有技术领导力

能够发现问题

能够提供解决方案和思路，并比较这些方案的优缺点

能够做出正确的技术决定

能够使用更简单更容易的方式来解决问题

能够提高代码或者软件的扩展性重用性和可维护性

能够正确管理团队

  • 让正确的人做正确的事，并发挥每个人的潜力

  • 可以提高团队的生产力和人效，找到最有价值的需求，用最少的成本实现，并不断提高自身和团队的标准

创新力：能够使用新的方法解决问题，追逐新的工具和技术

总是在提供解决思路和方案的人才是有技术领导力的人

# 如何做：

扎实的基础

非同一般的学习能力

坚持做正确的事情

不断提高对自己的要求

# 如何拥有技术领导力

  • 吃透基础技术，基础技术是上层技术共同的基础（建立自己的中台或者说是与通用技术能力体系）

    • 编程

      • C语言

      • 编程范式

        学好编程范式有助于培养你的抽象思维调试提高编程效率，提高程序的结构合理性，可读性和可维护性，降低代码冗余度，进而提升代码的运作效率

      • 算法和数据结构

        算法是编程中最最重要的东西

    • 系统

      • 计算机原理
      • 操作系统原理
      • 网络基础
      • 数据库原理
      • 分布式技术架构

  • 提高学习能力

    • 学习信息源
    • 与高手交流
    • 举一反三
    • 不怕困难
    • 开放的心态

  • 坚持做正确的事情

    • 提高效率的事情

    • 自动化的事情

    • 掌握前沿技术

    • 知识密集的事情

    • 技术驱动的事情

  • 高标准要求自己

    • google 的自我评分卡
    • 敏锐的技术嗅觉
    • 强调实践，学以致用
    • 永远在编程

# 如何成为被人愿意追随的leader

  • 帮助别人解决问题

  • 被人依赖

  • 赢得别人的信任

对于新人而言，并不是完全是别人相信你能够做到摸个事情，还有别人愿意向你打开心扉，和你说他心里做柔软的东西

  • 开发的心态以及倾向性价值观

开发的心态是指要有枪法的包容心，包容别人的观点以及价值观，但要有自己明确的观点

倾向性是指让别人明白你的立场，更清楚的指导你是怎么样的人，热不是捉摸不透，倾向性的价值观是别人是否愿意追随你的一个基础

  • 以身作则，终身写代码，这样你的方案才会更具有执行力和实践性

  • 保持热情和冲净

    正视问题，正视不足，正视错误，从中进行反思和总结得到更好的解决方案，不怕困难，迎难而上

  • 能够抓住重点，看透事务的本质

  • 描绘令人心动的方向，提供令人向往的环境

  • 甘当蒲石路，为他人创作机会
